Debug DNN & 2sxc Parameters

This just shows the configuration currently in use.

  1. Edition : {{ context.edition }}
    You would use this information for lazy-loading additional things from the correct build-folder (for example: if you needed to dynamically load some svg files).
  2. ApiEdition : {{ context.apiEdition }}
    This is automatically used in the 2sxc API calls, so that you can run a staging copy of your code without affecting public users or for A/B testing.
  3. Context information for HTTP
    These are automatically used in most http calls, because they tell the server what page the user is on.
    1. Page ID / Tab ID : {{ context.$2sxc.env.page() }}
    2. Sxc ID / Module ID : {{ context.sxc.id }}
    3. Anti Forgery Token : {{ context.$2sxc.env.rvt() }}
      This is automatically added to most http calls, for increased security. It changes with every page you load.
