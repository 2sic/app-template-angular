Hot-Loading: Develop locally, run live in DNN

You get up to 10x faster development when you can use live-compile and see the results in DNN from your local dev environment. Here's how it works:

  1. First you copy the app to your local drive (or clone the same repo to a local folder) and initialize everything (see checklist)
  2. Then you run npm run local in your console. This will start ng serve on localhost:4200 and whenever you save a code file, it recompiles in seconds.
  3. Our connector code in the _AngularIntegration.cshtml will now auto-load all the default bits so it just works!
